# クレセントアイルCE管理

### 使用方法
時刻を自動入力もしくは手動入力することで、ポットは60分後、CEは90分後の時刻が表示されるようになります。
**CEタイマー**は、90分が経過すると背景が赤くなります。
**ポットタイマー**は、"次回"時刻の5分前に自動で音が鳴り、背景が黄色くなります。(CEタイマーは音が鳴りません)
音の調整は上部の音量バーと音テストボタンから調整が可能です

CEタイマー右に、仕込みのチェックボックスがあります。仕込みが完了したと感じた時にチェックしてください。
**チェックボックス**は、タイマークリアと時刻入力ボタンを押した時にもリセットされます。

**タイマー共有**では時刻を出力して共有が可能です。共有された物をそのまま入力にコピペすることで自分のタイマーに反映できます。
※スペースの有無は問いませんが、名前部分もしくは時刻部分が崩れるとタイマーに反映されません。

わからない点・機能追加要望などがあれば前川店長(discord: t2maekawa_)までご連絡ください。

### v1.4.0-release (2025-12-23)

**追加機能**
新たに"銅 / 銀箱チェックマップ"を追加しました。ポットタイマーの上にあるボタンを押すと、新しいタブで開きます。
- 宝箱をクリックすると「取得済み」として記録されます。
- 取得状況、表示状態、ズーム倍率は自動的に保存されます。
- 銅箱／銀箱ボタンで表示の切り替えができます。
- ＋／− ボタンでマップを拡大・縮小できます。
- ズームリセットは倍率のリセット、全取得リセットは宝箱のチェック状態をリセットします。
